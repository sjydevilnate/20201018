        [
            {id:1, name: "슈퍼맨", power: 100},

        작업 순서
        1. Read 작업
            ==> 실행시 바인딩할 데이터
            [
                {id:1, name: "슈퍼맨", power: 100},
                {id:2, name: "아쿠아맨", power: 300},
                {id:3, name: "스파이더맨", power: 500},
                {id:4, name: "배트맨", power: 30},
            ]
        2. Delete 작업
            ==> 배열에서 삭제 ==> splice(배열의방번호, 개수) 사용.
        3. Update 작업
            ==> Up / Down 
            Up   ==> +100 증가 
            Down ==> -50 감소
        4. Insert 작업
            ==> "Add" 버튼을 클릭했을 때 추가 
            최대 id 값을 구하는 방법. 
            Object Array 에서 최대값 구하는 방법은 구글 검색을 이용한다.
            ==> 검색조건: javascript object array max
            방법1. .reduce()
            방법2. .map() 를 사용하는 방법
